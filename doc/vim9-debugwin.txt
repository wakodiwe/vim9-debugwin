INTRODUCTION                                                        *debugwin*

Simple plugin presenting a debug window when you need one.

COMMANDS                                                   *debugwin-commands*

:DebugWin text          Use the long                      *debugwin-:DebugWin* 
:D text                 Or the short command                     *debugwin-:D* 

USAGE                                                         *debugwin-usage*

Simply use :D 'some text' or :DebugWin 'some text' in the command line.
Or use it in your script:
>
        var g:superimportantvar = 'notsoimportant'
        DebugWin 'g:superimportantvar'
<
Or like this:
>
        var g:superimportantvar = 'notsoimportant'
        D 'g:superimportantvar'
<
